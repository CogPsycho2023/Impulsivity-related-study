This repository includes the key codes for individual gray-white matter contrast (GWC) extraction, resampling and surface-based functional connectivity (FC) analyses between individuals with low- and high-impulsivity scores
For statistics, it would be  better to see the tutorials in FSL (https://web.mit.edu/fsl_v5.0.10/fsl/doc/wiki/PALM(2f)Examples.html#Example_10:_Using_CIFTI_files)
